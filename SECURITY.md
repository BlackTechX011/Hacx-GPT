# Security Policy

## Supported Versions

| Version           | Supported          |
| ----------------- | ------------------ |
| current release   | :white_check_mark: |
| previous release  | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within HacxGPT, please do **NOT** create a public GitHub issue.

Instead, please report it via email to **security@blacktechx.com** (or your preferred contact method).

We will verify the issue and patch it as soon as possible.
